# OPCC Decision Records

Use a dated file named `YYYY-MM-DD-short-title.md` for a material source,
release, policy, security, or compatibility decision.

```markdown
# Decision: short title

## Context

## Decision

## Rationale

## Consequences

## Reopen when
```

Link the decision to the relevant issue, pull request, or release where it is
safe and useful to do so. Do not record private, restricted, or security
sensitive details here.
