# OPCC Licence and Attribution

OPCC code is released under the MIT licence. Each data layer retains its own
source licence, attribution, and provenance in its manifest; a layer may be
redistributed only when those terms allow it.

| Layer or component | Source | Licence / attribution | OPCC treatment |
| --- | --- | --- | --- |
| Package code and documentation | OPCC contributors | MIT; retain copyright notice | Redistributed with OPCC |
| NAR-derived observations | Statistics Canada, National Address Register | Open Government Licence - Canada; cite source and vintage | Primary source-qualified layer |
| Census geography | Statistics Canada, 2021 census geography | Open Government Licence - Canada; cite source and vintage | Used for geographic assignment |
| GeoNames supplementary points | GeoNames | CC BY 4.0; retain GeoNames attribution and source date | Separate supplementary layer |
| Canada Post, PCCF, PCCF+ | Canada Post / licensed products | Restricted or licensed | Never ingest, redistribute, or accept as contribution evidence |

Contributors must add a row or an equivalent manifest record for each proposed
source, including its stable identifier, licence, attribution text, retrieval
date, and redistribution assessment. When source terms conflict with this
document, the source terms control and the layer is not published until the
conflict is resolved.
